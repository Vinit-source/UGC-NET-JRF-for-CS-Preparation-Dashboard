# UGC NET Prep Dashboard

This project was developed to help aspirants streamline their preparation, stay organized, and actually enjoy the process.

Here’s a look at the Key Features:

1. Interactive Syllabus Tracker 🗺️: A comprehensive breakdown of Paper 1 and Paper 2 (Computer Science) with confidence markers. Track your preparation progress for every unit and sub-topic right from this main page!

2. Detailed Score Logging 🎯: A "Scores & Practice" section to log marks for units, topics, or sub-topics from your test series or general practice.

3. Integrated Study Journal & Calendar 📓: A dedicated space to write daily reflections and log study hours. You can even "tag" specific syllabus topics to your journal entries for better context.

4. Smart Connectivity 🔗: Clicking on a syllabus topic automatically pulls up your related journal entries, making it easy to track your learning evolution across any topic or sub-topic.

5. Schedule Your Week 📅: Based on the analysis from the Syllabus Tracker, you can intelligently schedule the right topics or sub-topics for your upcoming week.

6. Exam Countdown ⏳: A real-time tracker for the days remaining until the UGC NET JRF exam. It’s set to update the moment the official date for this season is announced!

7. AI-Driven Performance Analysis 🤖: The dashboard provides personalized, motivational feedback based on recent test scores, identifying your "strongholds" and exactly where you need to lean in further.

8. Privacy First 🛡️: Your data stays entirely on your computer and is never stored on any servers. You have full control over your information.

9. Export/Import for Migration 📤📥: Moving to a new device or browser? No problem. This feature ensures your data moves with you seamlessly, so you never lose your progress.

What’s next? 🛠️
Visual Charts 📊 to help users visualize their progress and score trends over time.

## Development Setup

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

### Prerequisites

- Node.js 18+ 
- npm, yarn, or pnpm

### Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
Create a `.env.local` file in the root directory and add your Gemini API key (optional, can also be provided in the app settings):
```env
NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build for Production

To build the application for production, run:
```bash
npm run build
```

Then, to start the production server:
```bash
npm start
```
